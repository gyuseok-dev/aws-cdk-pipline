# CDK TypeScript For 배포 자동화

TypeScript로 CDK 개발을 통해 codepipeline을 구성할 수 있는 예제 앱입니다.

cdk.json' 파일은 CDK 툴킷에 앱 실행 방법을 알려줍니다.

## 유용한 명령어

* `npm run build`   빌드
* `npm run watch`   변경사항 감지 모드
* `npm run test`    테스트 실행
* `cdk deploy`      배포 실행
* `cdk diff`        배포된 스택과 현재 상태를 비교
* `cdk synth`       Cloud Formation 템플릿 생성
